| Goal | Technique | File |

| Show past events | Flashback | temporal-narratives.md |
| Show future events | Flash-forward | temporal-narratives.md |

### By Mood

| Mood | Recommended Techniques |
|------|----------------------|
| **Happy/Optimistic** | High-key lighting, bright colors, smooth camera movement |
| **Dramatic/Tense** | Low-key lighting, handheld camera, hard shadows |
| **Mysterious** | Backlighting, non-linear narrative, low-key lighting |
| **Energetic** | Fast camera movement, whip pans, high contrast |
| **Calm/Serene** | Slow camera movement, soft lighting, natural light |
| **Sophisticated** | Rembrandt lighting, smooth steadicam, monochrome |
| **Playful** | Tilt-shift, forced perspective, bright colors |
| **Futuristic** | Colored lighting (cyan/blue), glitch effects, symmetrical framing |

### By Genre

| Genre | Recommended Styles |
|-------|-------------------|
| **Corporate** | Three-point lighting, static/smooth camera, professional |
| **Beauty/Fashion** | Butterfly lighting, slow motion, soft focus |
| **Tech/Innovation** | Colored lighting (blue), glitch effects, modern movement |
| **Luxury** | Low-key lighting, slow dolly, dramatic shadows |
| **Sports/Fitness** | Slow motion, tracking shots, high energy |
| **Documentary** | Natural lighting, handheld, realistic |
| **Action** | Fast camera movement, slow motion, high contrast |
| **Horror** | Low-key lighting, dutch angle, handheld |

### By Video Length

| Length | Recommended Approach |
|--------|---------------------|
| **5-10s** | Simple techniques, 1-2 transitions, clear message |
| **15s** | 2-3 techniques, simple narrative, focused |
| **30s** | Multiple techniques, story arc, varied pacing |
| **60s+** | Complex techniques, full narrative, varied styles |

## Common Combinations

### Professional Corporate
```
- Three-point lighting
- Static or smooth dolly camera
- Natural color palette
- Clean transitions
```

### High-Energy Sports
```
- Slow motion highlights
- Tracking shots
- High contrast lighting
- Fast cuts/whip pans
```

### Luxury Product
```
- Low-key dramatic lighting
- Slow dolly in
- Rack focus
- Monochrome or muted colors
```

### Tech Innovation
```
- Colored lighting (cyan/blue)
- Smooth camera movement
- Glitch effects (subtle)
- Modern, clean aesthetic
```

### Emotional Storytelling
```
- Natural lighting
- Handheld or steadicam
- Flashback sequences
- Soft, warm colors
```

### Creative/Artistic
```
- Symmetrical framing
- Creative techniques (tilt-shift, etc.)
- Unique color palettes
- Experimental transitions
```

## Prompt Engineering Shortcuts

### Camera Movement
```
Static: "locked-off tripod shot, camera remains still"
Pan: "smooth pan [left/right], camera rotates horizontally"
Dolly: "dolly [in/out], camera moves [forward/backward] on track"
Tracking: "tracking shot following [subject], maintains distance"
Handheld: "handheld camera, [subtle/active] natural movement"
```

### Lighting
```
Professional: "three-point lighting, balanced and clean"
Dramatic: "low-key lighting, deep shadows, high contrast"
Bright: "high-key lighting, bright and even, minimal shadows"
Natural: "natural [window/golden hour] lighting, authentic"
Moody: "colored lighting, [color] tones, atmospheric"
```

### Creative Effects
```
Slow-mo: "slow motion, [fps], dramatic detail"
Time-lapse: "hyperlapse, accelerated time, dynamic"
Frozen: "bullet time effect, subject frozen, camera rotates"
Miniature: "tilt-shift effect, narrow focus, toy-like"
Artistic: "black and white, high contrast, timeless"
```

## Common Mistakes to Avoid

### ❌ Don't Mix Conflicting Styles
```
Wrong: "Cheerful scene with dark, dramatic low-key lighting"
Right: "Cheerful scene with bright, high-key lighting"
```

### ❌ Don't Overuse Techniques
```
Wrong: "Bullet time + tilt-shift + glitch + slow motion all at once"
Right: "Choose 1-2 complementary techniques"
```

### ❌ Don't Ignore Motivation
```
Wrong: "Camera randomly moves for no reason"
Right: "Camera follows subject's movement naturally"
```

### ❌ Don't Mismatch Genre
```
Wrong: "Corporate video with handheld shaky cam"
Right: "Corporate video with stable, professional camera"
```

## Pro Tips

### 1. Start Simple
- Master basic techniques first
- Add complexity gradually
- Less is often more

### 2. Match to Brand
- Consider brand personality
- Maintain consistency
- Build visual identity

### 3. Consider Platform
- Instagram: Square/vertical, quick cuts
- YouTube: Horizontal, longer form
- TikTok: Vertical, fast-paced
- LinkedIn: Professional, clean

### 4. Test and Iterate
- Generate variations
- Get feedback
- Refine approach

### 5. Combine Thoughtfully
- Lighting + Camera movement
- Temporal + Transitions
- Creative + Traditional
- But don't overdo it
